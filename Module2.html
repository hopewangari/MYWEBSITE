Introduction to Embedded C

The second week began with an introduction to embedded systems, which are computing systems designed to perform dedicated functions within larger systems. We discussed:

  1. The definition and applications of embedded systems in everyday devices.
  2. Components of embedded systems: microcontrollers, sensors, actuators.
  3. The differences between general-purpose computing and embedded computing.
  4. Real-world examples in automation, medical devices, robotics, etc.

    5. This session emphasized the relevance of embedded systems in engineering fields.

Introduction to IDEs: Thonny, Arduino, Pico

We were introduced to development environments that facilitate embedded programming:

  1. Thonny: A beginner-friendly Python IDE, useful for scripting in microcontroller development.
  2. Arduino IDE: For programming microcontrollers like the Arduino Uno and ESP32 using simplified C/
  3. Pico SDK (C/C++): Introduction to programming Raspberry Pi Pico boards using low-level SDK tools.
